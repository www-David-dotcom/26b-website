# Branch: Refactor-React 
这个分支添加了react作为前端框架，把所有内容都进行了组建化处理，运行（开发测试）方式如下：

```bash
npm install
npm run dev
```

## Components
- Footer: 页脚组件
- Header：页面头部组件
- IntroScreen：引入页面组件
- NavBar：导航栏组件
- ThemeToggle：主题切换组件（暗黑/明亮模式）

## Pages
- Activities：活动页面
- Home：首页
- Contact：联系方式页面
- Journal：班级日志页面
- Introduction：班级介绍页面
- Wall: 留言墙页面
